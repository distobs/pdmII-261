O conceito de jogos de Linguagem foi usado por Ludwig Wittgenstein para
entender os diferentes usos da língua e das palavras. Para Wittgenstein, um
"jogo de linguagem" é o contexto em que se coloca um uso da língua, que, como
um jogo, teria regras às quais as palavras obedecem. Numa conversa, então, os
interlocutores estabelecem para si um jogo, ou uma forma de entender o que o
outro diz.

A exclamação "Chega!", por exemplo, pode ter diversos significados. Quando
despejo água no copo de um amigo e escuto "chega!", sei que meu amigo não quer
mais água. Quando alguém diz "Chega!" num protesto político, quer que as
exigências de sua causa sejam cumpridas. E, assim, a mesma palavra adquire
diversos significados a depender das "regras" do jogo.

Na educação, é possível utilizar os Jogos de Linguagem para fazer analogias
relacionadas aos conteúdos que estão sendo ensinados e para adequar os alunos a
diversas situações com que podem se deparar ao longo da vida.

Na Programação Orientada a Objetos, podemos dizer que as classes e as relações
entre elas podem simbolizar jogos de linguagem. Num programa que gerencia
empresas, quando falamos de "Empresa" e "Funcionário", falamos, de fato, das
_classes_ "Empresa" e "Funcionário", como forma de abstrair a vida real.
Exemplo:

```dart
class Funcionário {
        String nome;
        String cargo;
}

class Empresa {
        String nome;
        List<Funcionário> funcionarios;
}
```
